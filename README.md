# MiniLife 隐私政策 (Privacy Policy)

**生效日期：** 2025年12月30日

## 1. 概述 (Overview)
MiniLife（以下简称“本应用”）是一款由独立开发者张健开发的个人生活管理应用。我们非常重视您的隐私。本隐私政策旨在向您说明我们如何处理您的数据。

**核心承诺：我们不收集、不存储、不向任何第三方出售您的任何个人数据。** 您的所有数据（包括待办事项、习惯、日记、照片等）均存储在您的设备本地或您的个人 iCloud 账户中。

## 2. 数据存储与同步 (Data Storage & Sync)
* **本地存储**：您在 MiniLife 中创建的所有数据（如任务列表、习惯打卡记录、日记文本及图片）均使用 Apple 的 SwiftData/CoreData 技术存储在您的 iOS 设备本地数据库中。
* **iCloud 同步**：如果您开启了 iCloud 功能，您的数据将通过 Apple 的 iCloud 服务在您的设备之间同步。开发者无法访问您的 iCloud 数据。

## 3. 系统权限的使用 (System Permissions)
为了提供完整的功能，MiniLife 可能会请求以下系统权限。我们仅在必要时使用这些权限，且不会上传任何数据。

### 3.1 日历 (Calendar)
* **用途**：用于将您系统日历中的日程同步显示在 MiniLife 的任务列表中，方便您统一管理。
* **隐私**：我们仅读取您授权时间范围内的日历事件。这些数据仅用于本地显示和生成任务，不会发送至任何服务器。

### 3.2 面容 ID / 触控 ID (FaceID / TouchID)
* **用途**：用于“心语隐私锁”功能，保护您的日记内容不被他人查看。
* **隐私**：验证过程完全由 iOS 系统处理。App 仅接收“验证成功”或“验证失败”的结果，无法获取您的面部或指纹生物特征数据。

### 3.3 通知 (Notifications)
* **用途**：用于发送晨间唤醒、晚间复盘提醒以及习惯打卡提醒。
* **隐私**：所有通知均由本地触发（Local Notifications），不涉及远程推送服务。

### 3.4 相册与相机 (Photo Library & Camera)
* **用途**：用于在您的日记（心语）中添加照片。
* **隐私**：App 仅在您主动选择时读取特定的照片。照片数据仅存储在本地数据库中。

### 3.5 小组件 (Widgets)
* **用途**：为了在桌面显示数据，App 会将部分任务数据写入与小组件共享的本地容器（App Group）。
* **隐私**：该数据仅在您的设备内部共享，用于渲染 UI。

## 4. 第三方服务 (Third-Party Services)
本应用目前没有集成任何用于数据分析、广告投放的第三方 SDK（如 Google Analytics, Facebook Ads 等）。

## 5. 儿童隐私 (Children's Privacy)
本应用不针对 13 岁以下儿童设计，且不有意收集儿童的个人信息。由于我们不收集任何用户数据，因此也不存在涉及儿童数据的风险。

## 6. 隐私政策的变更 (Changes to This Policy)
我们要可能会不时更新我们的隐私政策。我们将通过在本页面上发布新的隐私政策来通知您任何更改。建议您定期查看本页面以获取任何变更信息。
